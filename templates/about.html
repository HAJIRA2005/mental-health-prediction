About This Project

Project Overview

This machine learning application predicts mental health status using classification algorithms. The system analyzes various lifestyle and health factors to provide insights into mental well-being.

Target Variable

Mental Health Status: Good, Moderate, or Poor

Dataset

Synthetic dataset with 500 samples for training

Analyzed Features

Sleep Duration

Average hours of sleep per night

Exercise Frequency

Days per week with physical activity

Social Interactions

Level of social engagement

Stress Level

Current stress assessment

Work Hours

Average working hours per day

Nutrition Score

Quality of dietary habits

Meditation

Daily mindfulness practice

Screen Time

Daily digital device usage

Machine Learning Models

Logistic Regression
Linear Model

Statistical method for binary/multiclass classification

Random Forest
Ensemble Method

Collection of decision trees for improved accuracy

SVM
Non-linear Model

Support Vector Machine for complex decision boundaries

Technology Stack

Backend

  • Flask Web Framework
  • Scikit-learn
  • Pandas & NumPy
Frontend
  • HTML5
  • CSS3 & Bootstrap
  • JavaScript
  • AJAX for predictions
